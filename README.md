# Práctica 8 - Automatización con Ansible

Este proyecto automatiza la configuración de 5 servidores Ubuntu usando Docker y Ansible.

##  Tecnologías usadas

- Docker
- Docker Compose
- Ansible
- WSL (Ubuntu)

##  ¿Qué hace el playbook?

- Actualiza los paquetes del sistema
- Crea el usuario `itla`
- Crea la carpeta `/home/itla/app`
- Crea el archivo `hola.txt` con contenido personalizado
- Instala `cowsay` y `htop`
